# 🔍 Visualizador de Perfil do GitHub

Uma aplicação web moderna e responsiva que permite aos usuários pesquisar perfis do GitHub e visualizar informações detalhadas como bio, seguidores, seguindo e os repositórios mais recentes.

## 🚀 Funcionalidades

- **Busca em Tempo Real:** Pesquisa rápida de usuários através da API oficial do GitHub.
- **Informações Detalhadas:** Exibição de avatar, nome, bio, número de seguidores e seguindo.
- **Listagem de Repositórios:** Visualização dos 10 repositórios mais recentes com estatísticas de estrelas, forks, watchers e linguagem principal.
- **Design Responsivo:** Adaptado para diferentes tamanhos de tela (desktop, tablet e mobile).
- **Animações Fluídas:** Interface dinâmica com feedback visual durante a navegação.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica do projeto.
- **CSS3:** Estilização avançada com animações e design responsivo (Mobile First).
- **JavaScript (ES6+):** Lógica da aplicação, manipulação do DOM e consumo de API Assíncrona (Fetch API).
- **API do GitHub:** Fonte de dados para perfis e repositórios.
- **Devicon:** Ícones representativos de tecnologias.
